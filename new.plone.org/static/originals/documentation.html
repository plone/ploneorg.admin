Recommended Books

  1. Content Management with Plone: Handbook for Authors and Editors End-user guide for Plone, recently updated for Plone 3.
  2. Practical Plone The most recent Plone book. Covers building robust, content-centric web applications with Plone 3.
  3. Theming Plone A guide to getting the most from Plone's theming system.
  4. Professional Plone Development The most recent Plone book. Covers building robust, content-centric web applications with Plone 3.
  5. Web Component Development with Zope 3 Zope 3 development. Gives a lucid introduction to the Zope 3 technologies available to Plone.
  6. Dive into Python The de facto standard for learning Python development.
  7. Expert Python Programming For the Python programmer who have mastered the basics.
  8. Learning jQuery A great guide to the Javascript framework used in Plone.
